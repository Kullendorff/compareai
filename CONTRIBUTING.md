# Contributing to AI Assistant

Thank you for your interest in contributing to this project! Here are some guidelines to help you get started.

## Ways to Contribute

- Bug reports and fixes
- Feature requests and implementations
- Documentation improvements
- User experience enhancements
- Testing and feedback

## Development Setup

1. Fork the repository
2. Clone your fork locally
3. Set up your API keys as described in the main README
4. Install dependencies: `pip install -r requirements.txt`
5. Run the application: `python app.py`

## Pull Request Process

1. Create a new branch for your feature or bugfix
2. Make your changes, following the existing code style
3. Add comments and documentation as necessary
4. Test your changes thoroughly
5. Create a pull request with a clear description of what you've done

## Code Style Guidelines

- Follow PEP 8 for Python code
- Use clear, descriptive variable and function names
- Include docstrings for all functions and classes
- Keep functions small and focused on a single task

## Questions?

If you have any questions or need help with anything, feel free to open an issue for discussion.

Thank you for contributing!
